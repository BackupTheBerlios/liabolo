Eingabe von Metadaten

Unter diesem Menupunkt können Sie die bereits bestehende Dokumente bearbeiten.

Eingabe von Metadaten

  • Klicken Sie das Feld an, welches Sie editieren möchten
  • Hinweis: Wenn Sie unsicher sind, welche Werte in welches Feld gehören, dann lassen Sie einfach den Mauszeiger auf dem leeren Feld stehen. Nach kurzer Zeit erscheint ein kleines Fenster mit einem Hilfetext (Tool-Tipp).
  • Bestätigen Sie Ihre Eingaben mit 'OK' oder brechen Sie die Bearbeitung ab
  • Hinweis: es werden nur Felder angezeigt, die im Formulareditor freigeschaltet wurden.

  • Quelle: Durch einen Klick auf den Button 'Quelle' können Sie ein Dokument durch das aktuelle referenzieren
