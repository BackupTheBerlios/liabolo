Arbeitsumgebung

Die Arbeitsumgebung ermöglicht Ihnen festzulegen, mit welchen Datenbanken Liabolo zusammenarbeiten soll. Grundsätzlich ist Liabolo immer mit einer mitgelieferten lokalen Datenbank verbunden. Alle Dokumente, die Sie selbst eingeben, werden in dieser Datenbank gespeichert. Liabolo kann aber auch mit Datenbanken im Netzwerk (Inter- oder Intranet) Kontakt aufnehmen. Damit erhalten Sie Zugriff auf die Daten anderer Nutzer oder auf zentrale Datenbanken, die gleichzeitig für mehrere Anwender bereitgestellt werden können.

Auswahlmöglichkeiten in der Arbeitsumgebung

  • Fachgebiete
  • Standorte
  • Indiv. Listen
  • Zwischenablage
  • Datenbank->Liabolo: Dient der Übertragung von Dokumenten aus dem Netzwerk in die lokale Liabolo Datenbank
  • Liabolo->Datenbank: Überträgt Daten aus der lokalen Liabolo Datenbank in das Netzwerk
  • Leeren: Hiermit können Sie den Inhalt der Zwischenablage löschen
Weitere Informationen zum Thema Arbeitsumgebung und Zwischenablage finden Sie im Benutzerhandbuch.
