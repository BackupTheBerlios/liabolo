Trefferliste/Suchergebnisse

Die Trefferliste zeigt die Dokumente an, die zu Ihrer Auswahl passen. Ob Sie ein Dokument suchen oder auf einen Fachbereich klicken, die Trefferliste zeigt die entsprechenden Einträge an. Mit diesem Fenster können Sie alle Aktionen wie 'Bearbeiten' oder 'Löschen' mit Dokumenten durchführen.

Das Ergebnis einer Suche in der Trefferliste

Hinweis: Markieren Sie einen Eintrag durch einen Mausklick. Mehrere Einträge können Sie durch das Setzen von Haken vor den Einträgen markieren. Klicken Sie dazu in die Kästchen vor jedem Eintrag, den Sie für den nächsten Bearbeitungschritt brauchen. Unter der Liste haben Sie die Möglichkeit, alle Dokumente mit einem Mausklick zu markieren.
  • Klicken Sie auf 'Neu', um ein Dokument hinzuzufügen
  • Klicken Sie auf 'Bearbeiten', um ein Dokument zu editieren
  • Klicken Sie auf 'Entfernen', um die markierten Dokumente nach einer Rückfrage zu löschen
  • Klicken Sie auf 'Zu Liste', um ein Dokument einer Individualliste hinzuzufügen
  • Klicken Sie auf 'Referenz', um ein Dokument hinzuzufügen, welches ein bestehendes Dokument referenziert
  • Klicken Sie auf 'Druckvorschau', um eine Druckansicht der gewählten Dokumente anzuzeigen
