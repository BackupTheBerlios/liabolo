Die Zwischenablage

Die Zwischenablage ermöglicht das Arbeiten auf einer Datenbank im Netzwerk/Internet. Nachdem Sie die Verbindung hergestellt haben, werden in allen Trefferlisten auch die Inhalte der entfernten Datenbanken angezeigt.

Um Objekte aus entfernten Datenbanken bearbeiten zu können. müssen Sie die die entsprechenden Einträge in Ihre Zwischenablage übertragen. Alle so kopierten Einträge werden in der Zwischenablage gesammelt, die Sie unter 'Verbindung/Zwischenablage' einsehen können.

Inhalt der Zwischenablage

Hinweis: Sie können nur Einträge in die Zwischenablage kopieren, die sich in einer entfernten Datenbank befinden. Entfernte Objekte erkennen Sie an Ihrer Signatur, diese enden NICHT auf '@localhost'. Die Endung dieser Einträge gibt den Server an, auf dem das Dokument gespeichert ist.

Unterschiedlichen Signaturen bei entfernten Datenbanken

Lokale Dokumente mit der Signaturendung '@localhost' können nicht in die Zwischenablage kopiert werden.
