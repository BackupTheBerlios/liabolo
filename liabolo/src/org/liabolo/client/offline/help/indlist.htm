Individualliste

Die Individualliste dient der Zusammenführung von Dokumenten, die für ein bestimmtes Projekt benötigt werden. Wenn Sie z.B. gerade an einer Diplomarbeit schreiben, können Sie alle Dokumente, die Sie referenzieren möchten, in einer Individualliste zusammenfassen.

Die Individuallisten im Überblick

  • Markieren Sie mit der Maus, die Einträge, die sie weiter bearbeiten möchten
  • Klicken Sie auf 'Neu', um eine Individualliste hinzuzufügen
  • Klicken Sie auf 'Bearbeiten', um eine Individualliste zu bearbeiten
  • Klicken Sie auf 'Inhalt', um die Dokumente anzuzeigen, die in den markierten Individuallisten eingetragen sind
