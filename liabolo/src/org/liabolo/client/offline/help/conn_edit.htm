Datenbankverbindung bearbeiten

Unter diesem Menupunkt können Sie bestehende Verbindungen zu Datenbanken im Netzwerk bearbeiten. Es erscheint eine Tabelle mit allen Verbindungen, die Liabolo gespeichert hat. Die letzte Spalte 'Aktiv' gibt an, ob zur Zeit eine Verbindung zu dieser Datenbank besteht. Hiermit können Sie überprüfen, auf welche Daten Liabolo überhaupt zugreifen kann.

Anzeige bestehender Verbindungen

  • Wählen Sie mit der Maustaste den Eintrag aus, den Sie bearbeiten möchten.
  • Klicken Sie auf 'Bearbeiten' um einen Eintrag zu verändern.
  • Bearbeiten Sie die Einträge in der Maske. Es gelten die gleichen Bedingungen wie für einen neuen Eintrag.
