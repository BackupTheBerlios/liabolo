Formulareditor

Mit dem Formulareditor können Sie entscheiden, welche Medadaten für ein Objekt angezeigt werden sollen. Liabolo unterstützt zur Zeit bis zu 15 Elemente für jedes Dokument, das Sie erfassen möchten. Die Elemente sind die Kernelemente des Dublin Core und erfüllen damit einen offenen Standard. Somit ist der Datenaustausch mit anderen Softwareprodukten problemlos möglich.

Formulareditor

  • Wählen Sie im Pulldownmenu einen bestehenden Medientyp
  • Aktivieren Sie die Felder, die Sie bei der Eingabe von Objekten bearbeiten möchten. Felder, die Sie vorerst nicht nutzen, können später jederzeit aktiviert werden.
  • Deaktivieren Sie die Felder, die sie nicht mehr nutzen möchten. Diese Felder werden zwar bei der Eingabe nicht mehr angezeigt, ihr Inhalt wird jedoch nicht gelöscht. Es werden keine Daten aus der Datenbank entfernt, die Anzeige wird lediglich übersichtlicher.
