Eingabe von Metadaten

Unter diesem Menupunkt können Sie die aktuell verbundene Datenbank mit Metadaten füllen

Eingabe von Metadaten

  • Wählen Sie einen Medientyp aus, es werden nur die bereits definierten Medientypen angezeigt. Klicken Sie auf das Drop-Down Feld, um die Liste der bestehenden Dokumenttypen angezeigt zu bekommmen.
  • Hinweis: Wenn Sie unsicher sind, welche Werte in welches Feld gehören, dann lassen Sie einfach den Mauszeiger auf dem leeren Feld stehen. Nach kurzer Zeit erscheint ein kleines Fenster mit einem Hilfetext.
  • Tragen Sie die Metadaten in die vorgegeben Felder ein
  • Hinweis: es werden nur Felder angezeigt, die im Formulareditor freigeschaltet wurden.
