Export von Metadaten

Liabolo bietet die Möglichkeit, Ihre Daten in ein anderes Format zu konvertieren. Für diese Umwandlung wird die "Extensible Stylesheet Language" (XSL) verwendet. Durch die Anpassung einer XSL Datei sind Sie so in der Lage, nahezu beliebige Ausgabeformate selbst zu erzeugen. Liabolo unterstützt zur Zeit den Export in das CSV-Format ('Comma Separated Value').

Export-Funktion für Individuallisten

  • Klicken Sie auf Datei/Exportieren
  • Wählen Sie die Individualliste, deren Inhalt Sie exportieren möchten
  • Klicken Sie auf 'Exportieren', um zum Dateidialog zu gelangen
  • Geben Sie einen Dateinamen an, unter dem Liabolo die Metadaten speichern soll
