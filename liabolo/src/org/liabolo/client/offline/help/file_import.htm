Import von Metadaten

Liabolo bietet die Möglichkeit, Excel-Tabellen zu importieren. Viele Anwender speichern Ihre Dokumente in einer Tabelle, um mit ihnen zu arbeiten. Die Tabellenkalkulation Excel ist in der Lage, viele verschiedene Dateiformate zu importieren. Weiterhin ist es mit freien Officepaketen wie Openoffice.org möglich, Excel Dokumente aus weiteren Formaten heraus zu erzeugen. Dadurch läßt sich die Formatvielfalt, die Liabolo importieren kann, noch erhöhen. Durch die Import Funktion für eine Excel-Tabelle lassen sich auf einfache Art und Weise bereits bestehende Daten in Liabolo importieren. Beispiel für eine Excel-Tabelle

Liabolo geht von einer zeilenorientierten Anordnung aus. Die Daten für jedes Dokument müssen nebeneinander angeordnet sein. Die Reihenfolge der Metadaten ist beliebig, Liabolo kann so konfiguriert werden, das die Eigenschaften der Dokumente den Feldern der Datenbank zugewiesen werden können.

Hinweis: Die Daten, die Sie importieren möchten, müssen sich auf der ersten Seite (Sheet 1) befinden. Tabellen auf anderen Seiten können nicht von Liabolo importiert werden.

Um die Daten einer Tabelle in Liabolo nutzen zu können, gehen Sie bitte folgendermaßen vor:

  • Klicken Sie auf 'Datei', dann 'Importieren'
  • Es erscheint ein Datei-Auswahl-Dialog. Navigieren Sie zur gewünschten Excel-Datei und öffnen Sie diese. Excel Dateien haben die Endung '.xls'.

    • Import-Dialog

  • Die Tabelle wird analysiert, einige Einstellungen müssen Sie allerdings noch selbst vornehmen, um eine einwandfreie Übernahme der Daten zu ermöglichen. Zu diesem Zweck erscheint ein Dialog, in dem Sie die Spalten aus Ihrer Tabelle den Feldern der Liabolo-Datenbank zuweisen können.

    • Zuweisungsdialog

  • Links sehen Sie die Spalten aus Ihrer Tabelle. Blättern Sie mit den Knöpfen '<<' und '>>' durch den Datenbestand. Dies ermöglicht Ihnen die eindeutige Identifikation Ihrer Felder und eine komfortable Zuweisung für den Import.
  • In der mittleren Spalte 'Map' können durch das Setzen oder Entfernen von Haken entscheiden, welche Spalten Ihrer Tabelle importiert werden sollen. Es werden nur Felder importiert, die mit einem Haken markiert sind!
  • Klicken Sie jetzt nacheinander auf jede Zeile in der Spalte 'Ziel'. Weisen Sie jeder Zeile ein Feld aus der Liabolo-Datenbank zu. Doppelte oder ungültige Zuweisungen werden rot hinterlegt. Unlogische Zuweisungen wie die doppelte Auswahl eines Feldes sind somit nicht möglich. Felder, deren Zuweisung korrekt ist, werden nicht farblich hervorgehoben. Im obigen Beispiel wurde das Feld 'Format' doppelt zugewiesen und damit rot hinterlegt.
  • Weisen Sie die Felder solange zu bis keine Felder mehr rot markiert sind.
  • Wählen Sie einen Medientyp unter 'Speichern als' aus, um die Dokumente zu klassifizieren. Alle Objekte Ihrer Tabelle werden diesem Medientyp zugewiesen.
  • Wählen Sie einen Fachbereich, in dem das Objekt gespeichert werden soll. Alle Elemente Ihrer Tabelle werden in diesem Fachbereich gespeichert.
  • Wählen Sie die Datenbank aus, in dem Ihre Dokumente gespeichert werden sollen, indem Sie auf das Feld neben 'Speichern in' klicken.
  • Bestätigen Sie den Import mit 'OK'.
  • Liabolo importiert nun Ihre Dokumente in die Datenbank.
