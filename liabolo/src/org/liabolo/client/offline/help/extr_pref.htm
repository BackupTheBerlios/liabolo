Einstellungen

Hier können sie generelle Einstellungen von Liabolo vornehmen. Diese wirken sich sofort auf sämtliche Funktionen aus. Die Änderung der Sprache erfordert allerdings einen Neustart von Liabolo.

Einstellungen

  • 'Toolbar anzeigen': hiermit können Sie die Symbolleiste am linken Fensterrand von Liabolo ein- und ausschalten. Diese dient dem schnellen Zugriff auf wichtige Funktionen.
  • 'Warnungen anzeigen': mit dieser Funktion können Warnungen bei der Eingabe und Speicherung abgeschaltet werden. Fortgeschrittene Nutzer, die mit der Bedienung von Liabolo vertraut sind, können hiermit lästige Bestätigungen häufig auftretender Funktionen abschalten.
  • 'Sprache wählen': Liabolo unterstützt die Sprachen Deutsch und Englisch. Nach einer Änderung muss Liabolo neu gestartet werden.
  • 'Automatischer Verbindungsaufbau': diese Funktion legt fest, das Liabolo sofort nach dem Start eine Verbindung zu einem Datenbankserver im Netzwerk hmen soll. Diese Einstellung wird empfohlen, wenn eine permanente Anbindung an das Internet vorhanden ist.
