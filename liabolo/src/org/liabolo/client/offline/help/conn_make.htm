Datenbankverbindung herstellen/trennen

Unter diesem Menupunkt können Sie die Verbindung zur Datenbank herstellen und trennen.

Liabolo mit bestehender Verbindung

Hinweis: Beachten Sie den Globus neben dem Menupunkt 'Verbinden' und in der Symbolleiste. Ein Globus in grüner Farbe zeigt an, das eine Verbindung zur gewünschten Datenbank besteht. Ist das Symbol rot, so besteht zur Zeit keine Verbindung zu einer entfernten Datenbank.
