Sicherung/Wiederherstellung der Datenbank

Mit diesem Menupunkt können Sie ihre lokale Datenbank auf der Festplatte oder einem anderen Medium sichern und auch wieder herstellen. Hinweis: die Dialoge sehen sich sehr ähnlich, beachten Sie die beiden Registerkarten an der oberen Seite des Fensters.

Sicherung der Datenbank

Registerkarte Sicherung

  • Klicken Sie auf die Registerkarte 'Sichern' am oberen Fensterrand
  • Wählen Sie einen Ordner aus, in dem die Datenbank gespeichert werden soll. angezeigt.
  • Geben Sie einen Namen für die Sicherung ein
  • Bestätigen Sie Ihre Auswahl mit 'Save'

    • Hinweis: Liabolo legt ein Verzeichnis mit dem Namen an, den Sie gewählt haben. In diesem Verzeichnis finden Sie ein Unterverzeichnis mit Namen: 'db'. In diesem Unterverzeichnis befindet sich die Datei: '__contents__.xml' Diese Datei enthält alle Informationen über die Sicherung Ihrer Daten. Wenn Sie mehrere Sicherungen erstellen möchten, so speichern Sie die Datenbank in unterschiedlichen Verzeichnissen.

Wiederherstellung der Datenbank

Registerkarte Wiederherstellung

  • Klicken Sie auf die Registerkarte 'Wiederherstellen' am oberen Fensterrand
  • Wählen Sie einen Ordner aus, in dem sich die Sicherung der Datenbank befindet
  • Wählen Sie das Unterverzeichnis mit dem Namen 'db' aus
  • Wählen Sie die Datei '__contents__.xml' aus
  • Bestätigen Sie Ihre Auswahl mit 'Open'

Hinweis: Liabolo spielt die gesicherten Daten wieder in die Datenbank ein. Sie müssen die Datei '__contents__.xml' auswählen, um eine Sicherung wieder herstellen zu können. Wenn Sie mehrere Sicherungen angelegt haben, so navigieren Sie bitte in das Verzeichnis, in dem sich die gewünschte Sicherung befindet. Der Dateiname unterschiedlicher Sicherungen ist jedoch immer gleich.
