Dokumente einer Individualliste hinzufügen

Unter diesem Menupunkt können Sie Dokumente einer oder mehreren Individuallisten zuordnen. Mit dieser Funktion können Sie Dokumente zusammenführen, die Sie für ein Projekt (z.B. eine Diplomarbeit) benötigen. Diese zusammen gehörenden Dokumente werden in einer Individualliste gespeichert.

Auswahl der bestehenden Individuallisten

  • Markieren Sie eine Individualliste, um das Dokument aufzunehmen
  • Mehrere Individuallisten können markiert werden, indem Sie beim Klicken die 'STRG'-Taste gedrückt halten
  • Durch einen Klick auf den Kasten 'Alle markieren' können Sie ein Dokument allen bestehenden Individuallisten zuordnen
  • Bestätigen Sie Ihre Auswahl mit 'OK' oder verlassen Sie den Dialog durch 'Abbrechen'
