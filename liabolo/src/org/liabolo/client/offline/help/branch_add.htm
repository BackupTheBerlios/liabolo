Hinzufügen von Fachbereichen

Unter diesem Menupunkt können Sie selbst definierte Fachbereiche anlegen.

Hinzufügen von Fachbereichen

  • Geben Sie einen Namen für das Fachgebiet ein.
  • Geben Sie eine Kurzform für das Fachgebiet ein. Die Abkürzung muss aus GENAU drei Buchstaben bestehen und geht direkt in die Signatur ein.
  • Bestätigen Sie Ihre Eingabe mit 'ok'.
