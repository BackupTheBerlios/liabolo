Hinzufügen einer Individualliste

Die Individualliste speichert die Dokumente, die Sie individuell zusammenfassen bzw. gruppieren wollen. In dieser Liste können sie die Objekte sammeln, die später z.B. in einem Literaturverzeichnis erscheinen sollen oder zu einer Diplomarbeit gehören.

Die Icons

  • Geben Sie einen Namen für die Individualliste ein
  • Geben Sie eine Beschreibung für die Individualliste ein
  • Bestätigen Sie Ihre Eingabe mit 'ok' oder brechen Sie den Dialog ab
