Unter diesem Menupunkt können Sie bereits angelegte Fachbereiche bearbeiten.

Bearbeiten von Fachbereichen

  • Ändern Sie den Namen für den Fachbereich.
  • Bestätigen Sie Ihre Eingabe mit OK oder brechen Sie die Bearbeitung ab.
Hinweis: Das Kürzel für einen Fachbereich können Sie aus technischen Gründern nicht mehr verändern.
