Suche nach Metadaten

Liabolo bietet zwei Möglichkeiten, in der Datenbank zu suchen: Suchen und Blättern.

Suche

Mit der Suchfunktion können Sie selbst ein Stichwort eingeben, nach welchem in der Datenbank gesucht werden soll.

Suchdialog

  • Markieren Sie die Felder, in denen gesucht werden soll, mit einem Haken.
  • Geben Sie einen Suchbegriff ein.
  • Klicken Sie auf Senden, um die Suche zu starten.
  • Liabolo präsentiert eine Tabelle, in der alle Dokumente aufgeführt werden, die den Suchbegriff enthalten.

Blättern

Das Blättern ermöglicht das komfortable Surfen in den eingegebenen Daten. Liabolo gibt eine Liste der Fachbereiche aus.

Blättern-Dialog

  • Markieren Sie alle Fachbereiche, die angezeigt werden sollen. Markieren Sie einen Fachbereich mit der Maus. Halten Sie die STRG-Taste gedrückt, wenn Sie weitere Fachbereiche markieren möchten. Setzen Sie einen Haken vor 'Alle markieren', um alle Fachbereiche auszuwählen.
  • Bestätigen Sie Ihre Auswahl mit Senden, Liabolo zeigt alle Dokumente an, die sich in den gewählten Fachbereichen befinden.
