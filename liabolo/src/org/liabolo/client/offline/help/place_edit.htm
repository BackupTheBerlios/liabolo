Bearbeiten von Standorten

Unter diesem Menupunkt können Sie bereits angelegte Standorte bearbeiten oder löschen.

Bearbeiten von Standorten

  • Wählen Sie aus der Liste einen Standort aus.
  • Klicken Sie auf 'Bearbeiten'
  • Verändern Sie die gewünschten Werte
  • Speichern Sie Ihre Änderungen
  • Wählen Sie 'Entfernen', um ein Fachgebiet nach einer Rückfrage zu löschen.
