	\section{Leistungsmerkmale}
	\label{sec:features}
	Liabolo ist eine Metadatenverwaltung. Bei der Erfassung Ihrer Dokumente haben Sie die Möglichkeit, alle Eigenschaften wie zum Beispiel Titel, Autor oder  Erscheinungsjahr einzugeben. Doch Liabolo bietet noch mehr als das einfache Anlegen von Informationen über Ihre Dokumente:
	
	\begin{itemize}
	\item{Datensätze können jederzeit angelegt, bearbeitet oder gelöscht werden}
	\item{Intuitive Bedienbarkeit durch grafische Benutzeroberfläche}
	\item{Gemeinsame Nutzung einer Datenbank im Inter-/Intranet}
	\item{Import von Excel-Tabellen Excel für die schnelle Übernahme bestehender Datensammlungen}
	\item{Exportfunktion ausgewählter Daten für die Erstellung von Literaturverzeichnissen}
	\item{Sicherungs-/Wiederherstellungsfunktion für die lokale Datenbank}
	\item{Umfangreiche Online-Hilfe für die Unterstützung bei der täglichen Arbeit}
	\item{Deutsche und englische Sprachunterstützung}
	\end{itemize}

	\section{Systemanforderungen}
	\label{sec:system-requirements}
	\index{Systemanforderungen}
	Liabolo wurde in der plattformunabhängigen Programmiersprache Java geschrieben. Java ist ein Produkt der Firma Sun Microsystems \cite{sun}.\index{Java}Die Systemvoraussetzung für den Einsatz von Liabolo ist die Installation einer Java Laufzeitumgebung (JRE) in der Version 1.4.2. Sie können diese Laufzeitumgebung kostenlos im Internet unter folgender Adresse herunterladen:
	\url{http://java.sun.com/j2se/1.4.2/download.html}\\
	Liabolo kann weiterhin überall dort eingesetzt werden, wo die Installation der Java Laufzeitumgebung gelingt:\\
	Softwarevoraussetzungen:\\
	 Windows 98 (SE)/NT/2000/2003/ME/XP, Solaris 7/8/9/64 Bit, Linux, MacOS, BSD\\
	 
	 Hardwarevoraussetzungen: bitte entnehmen Sie die Hardwarevoraussetzungen für die jeweiligen Architekturen den Installationsanweisungen für die entsprechende Plattform: http://java.sun.com/j2se/1.4.2/jre/install.html\\
	 Liabolo wurde auf folgenden Betriebsystemen erfolgreich getestet: Windows 98 SE/2000/XP, Linux, MacOS und FreeBSD. Ob auf Ihrem Rechner eine geeignete Java-Installation vorliegt können Sie folgendermaßen überprüfen:
	 \begin{itemize}
	 \item{Öffnen Sie die Eingabeaufforderung/Console}
	 \item{Geben Sie 'java -version' ein}
	 \item{Überprüfen Sie die Bildschirmausgabe auf die Version der Java-Umgebung. Liabolo benötigt mindestens die Version 1.4.0}
	 \item{Wenn der Aufruf eine Fehlermeldung erzeugt, ist keine Java-Laufzeitumgebung installiert}
	 \end{itemize}
	 
	 Eine korrekte Installation von Java erzeugt z.B. folgende Ausgabe:
	 \begin{verbatim}
	$ java -version
	java version "1.4.2-p6"
	Java(TM) 2 Runtime Environment,
	Standard Edition (build 1.4.2-p6-06_jan_2004_12_12)
	Java HotSpot(TM) Client VM (build 1.4.2-p6-06_jan_2004_12_12, mixed mode)
	 \end{verbatim}
	 Die Lauffähigkeit von Liabolo mit älteren Versionen als 1.4.2 wurde nicht überprüft. Bitte verwenden sie eine aktuelle Version, wenn möglich.

	Dieser Abschnitt beschreibt die Installation von Liabolo, nachdem Sie die obigen Installationsvoraussetzungen erfüllt haben.
	\begin{itemize}
	\item{Laden Sie Liabolo aus dem Internet unter folgender Adresse herunter:
	http://developer.berlios.de/projects/liabolo/}
	\item{Die Datei heißt Liabolo-Vx.x.zip und zeigt an, das es sich um ein Archiv im ZIP-Format handelt.}
	\item{Wechseln Sie mit dem Dateimanager in das Verzeichnis, in dem Sie Liabolo gespeichert haben}
	\item{Entpacken Sie das Zip-Archiv in ein Verzeichnis Ihrer Wahl}
	\item{Starten Sie Liabolo mit einem Klick auf 'start.bat' (Windows) oder 'start.sh' (Unix/Linux)}
	\item{\underline{Hinweis für Unix/Linux:} sollte die Datei start.sh Liabolo nicht starten, so ist die Datei möglicherweise nicht als ausführbar gekennzeichnet. Öffnen Sie ein Terminal und wechseln Sie in das Verzeichnis, in dem sich die Datei 'start.sh' befindet. Geben Sie folgenden Befehl ein: 'chmod +x start.sh'}
	\item{Liabolo sollte jetzt gestartet werden können}
	\end{itemize}

		\subsection{Metadaten}
		\label{sec:metadata}
		\index{Metadaten}
		Liabolo ist eine Metadatenverwaltung. Bei der Eingabe werden alle zugehörigen Informationen (z.B. Titel, Autor) über ein Dokument in einer Datenbank gespeichert. Liabolo unterstützt zur Zeit bis zu 15 Elemente für jedes Dokument, das Sie erfassen möchten. Die Elemente sind die Kernelemente des Dublin Core \index{Dublin Core}und erfüllen damit einen offenen Standard. Somit ist der Datenaustausch mit anderen Softwareprodukten problemlos möglich. Liabolo speichert die folgenden 15 Metadaten für jedes Objekt in der Datenbank: 
		\begin{itemize}
		\item{Title: Dokumentname}
		\item{Creator: Verantwortlicher für den Dokumentinhalt}
		\item{Subject: Thema und Schlüsselworte}
		\item{Description: Inhaltsbeschreibung}
		\item{Publisher: Verleger/Herausgeber}
		\item{Contributor: Coautoren/Unterstützer}
		\item{Date: Datum der Veröffentlichung}
		\item{Type: Kategorie oder Genre}
		\item{Format: Physikalischer oder digitaler Dokumenttyp}
		\item{Identifier: Signatur oder URI}
		\item{Source: Quelle oder ursprüngliches Gesamtwerk}
		\item{Language: Sprache}
		\item{Relation: Referenzierung}
		\item{Coverage: Ablageort}
		\item{Rights: Rechteinhaber}
		\end{itemize}
		Mit diesen Informationen lassen sich alle Arten von Dokumenten (Buch/Video/Musik) beschreiben. Weitere Informationen zum Dublin Core Standard finden Sie im Internet unter: \cite{dublincore}.

		\subsection{Zwischenablage}
		\label{sec:clipboard}
		Die Zwischenablage ermöglich das Arbeiten mit Dokumenten, die sich in entfernten Datenbanken befinden. Nachdem Sie die Verbindung hergestellt haben, werden in allen Trefferlisten auch die Inhalte der entfernten Datenbanken angezeigt. Durch die Zwischenablage sind Sie in der Lage, mit Dokumenten zu arbeiten, die sich nicht in Ihrer lokalen Datenbank befinden.\\
		\underline{Beispiel:} auf einer bevorstehenden Zugreise möchten Sie an Ihrer Publikation weiter arbeiten. Sie benötigen allerdings Verweise auf Dokumente, die Sie nicht in ihrer lokalen Datenbank gespeichert haben. Mit Liabolo können Sie die fehlenden Dokumente aus entfernten Datenbanken in die Zwischenablage kopieren und den Rechner ausschalten. Wenn unterwegs dann keine Verbindung zum Netzwerk besteht, können Sie dennoch mit den Dokumenten in der Zwischenablage arbeiten.\\
		\begin{figure*}[!htb]
                  \begin{center}

                  \end{center}

Entfernte Objekte erkennen Sie an Ihrer Signatur, diese enden NICHT auf '@localhost'. Die Endung dieser Einträge gibt den Server an, auf dem das Dokument gespeichert ist.

		\subsection{Neu}
		\label{sec:connection-new}
		Unter diesem Menupunkt können Sie neue Verbindungen zu Datenbanken im Netzwerk aufnehmen. Für die Verbindung werden verschiedene Angaben benötigt. Liabolo ist in der Lage, mit unterschiedlichen Datenbanken eine Verbindung aufzubauen. Hierfür ist die Angabe eines passenenden Treibers erforderlich.
		\begin{figure*}[!htb]
                  \begin{center}
                    \includegraphics[scale=0.5]{images/conn_new.png}
                  \end{center}
	          \caption{Verbindung hinzufügen}
        	  \label{fig:conn-new}
               	\end{figure*}\par
		
		\begin{itemize}
		\item{Geben Sie einen Namen für die Verbindung ein. Diesen Namen können Sie frei wählen, er dient zur Unterscheidung der einzelnen Verbindungen.}
		\item{Weiterhin geben Sie die URI ein, unter der die Datenbank zu erreichen ist. Sie müssen die Art der Datenbank, die Adresse und den Port angeben, auf dem die Datenbank zu erreichen ist. Durch diese drei Angaben kann eine Datenbank eindeutig bezeichnet werden und unterscheidet sich damit von allen anderen möglichen Datenbanken.}
		  \begin{itemize}
		  \item{Geben Sie die Art der Datenbank an, gefolgt von einem Doppelpunkt und zweimal dem Symbol Slash ('/'). Für eine Datenbank vom Typ eXist lautet dieser Teil: 'exist://'}
		  \item{Weiterhin wird die Adresse benötigt, unter der die Datenbank zu erreichen ist. Sie können den Namen oder die IP-Adresse eines Computers angeben. Der Name setzt sich im Allgemeinen aus Host, Domain und Top-Level-Domain zusammen. Ein vollständiger Name lautet z.B.: 'p30srv.wiwi.uni-oldenburg.de'}
		  \item{Weiterhin wird die Adresse benötigt, unter der die Datenbank zu erreichen ist. Sie können den Namen oder die IP-Adresse eines Computers angeben. Der Name setzt sich im Allgemeinen aus Host, Domain und Top-Level-Domain zusammen. Ein vollständiger Name lautet z.B.: 'p30srv.wiwi.uni-oldenburg.de'}
		  \item{Abschließend wird der Port benötigt, auf dem die Datenbank auf Anfragen antwortet. Dieser Port ist eine Zahl zwischen 1 und 65535 und abhängig von der Installation und der Art der Datenbank. In diesem Beispiel erwartet die Datenbank eXist Anfragen auf dem Port 8081. Der Port muss an die Adresse angehängt werden, nachdem ein Doppelpunkt als Trennzeichen eingefügt wurde.}
		  \end{itemize} 
		\item{Fügen Sie die Art der Datenbank, gefolgt von den Trennzeichen '://', die Adresse des Servers gefolgt von einem Doppelpunkt und den Port zu einem Ausdruck zusammen und tragen Sie ihn unter DB-URI ein. Ein korrektes Beispiel lautet: 'exist://p30srv.wiwi.uni-oldenburg.de:8081'}
		\item{Weiterhin wird ein Benutzername benötigt, der den Zugriff auf die angegebene Datenbank ermöglicht.}
		\item{Geben Sie das zum Benutzernamen passende Kennwort ein. }		
		\item{Geben Sie den entsprechenden Treiber für die Datenbank ein. Dieser Treiber ermöglicht den korrekten Zugriff auf die Datenbank. Liabolo kann so auf verschiedene Datenbankarchitekturen zugreifen. Der korrekte Treiber für eine eXist Datenbank lautet: 'org.exist.xmldb.DatabaseImpl'}		
		\item{Bestätigen Sie Ihre Angaben mit 'OK' oder verlassen Sie den Dialog mit 'Abbrechen', um Ihre Änderungen nicht zu speichern.}
		\end{itemize}

	\section{Extras}
	\label{sec:extra}
		\subsection{Formulareditor}
		\label{sec:extra-formeditor}
		\index{Formulareditor}
		Mit dem Formulareditor können Sie entscheiden, welche Medadaten für ein Objekt angezeigt werden sollen. Liabolo unterstützt zur Zeit bis zu 15 Elemente für jedes Dokument, das Sie erfassen möchten. Die Elemente sind die Kernelemente des Dublin Core \cite{dublincore} und erfüllen damit einen offenen Standard. Somit ist der Datenaustausch mit anderen Softwareprodukten problemlos möglich.
		\begin{figure*}[!htb]
                  \begin{center}
                    \includegraphics[scale=0.5]{images/form_edit.png}
                  \end{center}
	          \caption{Der Formulareditor}
        	  \label{fig:form-edit}
               	\end{figure*}\par
		Wählen Sie im Pulldownmenu einen bestehenden Medientyp. Aktivieren Sie die Felder, die Sie bei der Eingabe von Objekten bearbeiten möchten. Felder, die Sie vorerst nicht nutzen, können später jederzeit aktiviert werden. Deaktivieren Sie die Felder, die sie nicht mehr nutzen möchten. Diese Felder werden zwar bei der Eingabe nicht mehr angezeigt, ihr Inhalt wird jedoch nicht gelöscht. Es werden keine Daten aus der Datenbank entfernt, die Anzeige wird lediglich übersichtlicher.

		\subsection{Einstellungen}
		\label{sec:extra-preferences}
		Hier können sie generelle Einstellungen von Liabolo vornehmen. Diese wirken sich sofort auf sämtliche Funktionen aus. Die Änderung der Sprache erfordert allerdings einen Neustart von Liabolo. 
		\begin{figure*}[!htb]
                  \begin{center}
                    \includegraphics[scale=0.5]{images/preferences.png}
                  \end{center}
	          \caption{Einstellungen}
        	  \label{fig:preferences}
               	\end{figure*}\par
		\begin{itemize}
		\item{'Toolbar anzeigen': hiermit können Sie die Symbolleiste am linken Fensterrand von Liabolo ein- und ausschalten. Diese dient dem schnellen Zugriff auf wichtige Funktionen.}
		\item{'Warnungen anzeigen': mit dieser Funktion können Warnungen bei der Eingabe und Speicherung abgeschaltet werden. Fortgeschrittene Nutzer, die mit der Bedienung von Liabolo vertraut sind, können hiermit Bestätigungen häufig auftretender Funktionen abschalten.}
		\item{'Sprache wählen': Liabolo unterstützt die Sprachen Deutsch und Englisch. Nach einer Änderung muss Liabolo neu gestartet werden. Liabolo kann leicht durch das Hinzufügen einer Datei an weitere Sprachen angepasst werden.}
		\item{'Automatischer Verbindungsaufbau': diese Funktion legt fest, das Liabolo sofort nach dem Start eine Verbindung zu einem Datenbankserver im Netzwerk hmen soll. Diese Einstellung wird empfohlen, wenn eine permanente Anbindung an das Internet oder Netzwerk vorhanden ist.}
		\end{itemize} 
		


		
	
	\section{Hilfe}
	\label{sec:help}
	\index{Hilfe}
		\subsection{Inhalt}
		\label{sec:help-content}
		Über diesen Punkt erreichen Sie die Hilfe von Liabolo, die Sie gerade lesen. :-)
		\subsubsection{Tool-Tips}
		Liabolo verfügt nicht nur über diese Online-Hilfe, sondern unterstützt Sie auch direkt bei der Eingabe in den meisten Dialogen. Wenn Ihnen an einer Stelle unklar ist, was Sie in ein bestimmtes Feld eintragen sollen, so können Sie sich durch einen Tool-Tip helfen lassen: bewegen Sie die Maus auf das Feld, dessen Bedeutung Ihnen unklar ist. Nach kurzer Zeit erscheint ein kleines Fenster mit einem Hilfetext, der Ihnen die Bedeutung des Feldes erläutert. 
		\subsubsection{Rechte Maustaste}
		Falls Ihnen nicht klar ist, welche Funktionen Sie an einer bestimmten Stelle ausführen können, so drücken Sie die rechte Maustaste. In einem Fenster erscheinen die Menupunkte, die zu diesem Zeitpunkt sinnvoll sind.
		\subsubsection{Navigation in der Hilfe}
		Die Online-Hilfe ist in der Seitenbeschreibungssprache HTML verfasst und verhält sich ähnlich wie der Seitenaufruf im Internet. Auf jeder Seite der Hilfe finden Sie am oberen Rand einen Link zum '[index]', der Sie zum Inhaltsverzeichnis der Hilfe führt. Weiterhin bieten die Icons am oberen Fensterrand folgende Funktionen: 
		\begin{figure*}[!htb]
                  \begin{center}
                    \includegraphics[scale=0.5]{images/help_buttons.png}
                  \end{center}
	          \caption{Die Icons der Online-Hilfe}
        	  \label{fig:help_buttons}
               	\end{figure*}\par
		\begin{itemize}
		\item {Klicken Sie auf das Symbol 'Pfeil nach links', um immer zum Inhaltsverzeichnis der Hilfe zu gelangen}
		\item {Klicken Sie auf das Symbol 'Drucker', um die Online-Hilfe auszudrucken
		\item {Klicken Sie auf das Symbol 'geöffnete Tür', um die Online-Hilfe zu} verlassen}
		\end{itemize}
